How to Use This Site

  • About eXo Platform Documentation
    • Administrator Guide: This guide describes how to get started with eXo Platform, specifically for:
      • Developers who want to know how to leverage eXo Platform in their customer projects.
      • System Administrators who want to use, deploy and manage eXo Platform system in their enterprise.

      This document will guide you through the most important tasks for eXo Platform administration and management. At the end, you will be able to install, configure, migrate, and manage your eXo Platform system.

    • Developer and Reference Guides: These guides are intended for systems integrators who want to know how to leverage eXo Platform in their customer projects and IT enterprises who need to customize and deploy their portal.

      • This guide introduces the eXo architecture, and shows developers how to perform some of the most common tasks needed for working with the eXo Platform. It also serves as an entry point for the Reference Guide, which provides more in-depth technical details about eXo Platform 3.0. As a result, you will be able to customize your own portal with eXo Platform.

    • User Guides:
      • Gatein

        • GateIn portal framework is an open source project developed jointly by Red Hat / JBoss. It provides a powerful framework for developing portlets and other web-based user interface.

      • eXo Content

        • eXo Content combines several eXo open source project to provide eXo Platform 3 with comprehensive Enterprise Content Management (ECM) capabilities. Quickly build websites, add document management and workflow to portal-based apps, and much more.

      • eXo Social

        • eXo Platform 3 includes a framework for building gadgets that can display and mash-up actitivy information for contacts, social networks, applications and services.

      • eXo Collaboration

        • Add rich collaboration and communication features like Chat, Mail, Address Books and Calendars to your portal-based web apps.

      • eXo Knowledge

        • Extend your eXo-based web apps with knowledge management functionality like Forums, Answers and FAQs.

  • Navigating

    • Browse topics in the Contents frame () on the left. Click on a topic to have it displayed. Use the Back and Forward buttons to navigate within the history of viewed topics.

  • Searching

    • To quickly locate topics on a particular subject in the documentation, enter a query in the Search field. Use the Search frame () to display the Search view. You can narrow the scope of your search by selecting only the sections you are interested in.

  • Synchronizing

    • Clicking the Show in Table of Contents button () will select that topic in the navigation tree. The Link with Contents button () keeps the navigation tree synchronized to the current topic.
